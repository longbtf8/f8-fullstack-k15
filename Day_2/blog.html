Hướng dẫn tạo ứng dụng Todo List đơn giản với HTML, CSS và JavaScript

Bước 1: Tạo cấu trúc thư mục

Trước khi lao vào code, bạn đã chuẩn bị folder chưa? Mình gợi ý tạo một thư mục tên todo-app, trong đó có ba file:

  • index.html
  • styles.css
  • app.js

Vậy là xong folder. Giờ thì… có file để mở code chưa?

Bước 2: Viết HTML

Mô tả trước khối code

Đây là phần khung xương, bạn đặt form để nhập task, một <ul> để hiển thị danh sách, rồi link đến CSS/JS. 

<!-- index.html -->
<!DOCTYPE html>
<html lang="vi">
<head>
 <meta charset="UTF-8" />
 <meta name="viewport" content="width=device-width, initial-scale=1.0" />
 <title>Todo List App</title>
 <link rel="stylesheet" href="styles.css" />
</head>
<body>
 <div class="container">
   <h1>My Todo List</h1>
   <form id="todo-form">
     <input type="text" id="todo-input" placeholder="Nhập việc cần làm..." />
     <button type="submit">Thêm</button>
   </form>
   <ul id="todo-list"></ul>
  </div>
  <script src="app.js"></script>
</body>
</html>

Giải thích sau khối code

  1. <form id="todo-form">để bắt sự kiện submit khi người dùng nhấn “Thêm”.
  2. <input id="todo-input">nhận text của todo mới.
  3. <ul id="todo-list">chứa các <li>đại diện cho từng task.
  4. Cuối trang chèn app.js để gắn logic JavaScript.

Bước 4: Viết JavaScript xử lý thêm, hiển thị và xóa

Mô tả trước khối code

Ở đây mình:

  1. Bắt form submit, ngăn reload page.
  2. Lấy value từ #todo-input, push vào mảng todos.
  3. Hàm renderTodos() tạo DOM cho mỗi task và gắn sự kiện xóa.
// app.js
const form = document.getElementById('todo-form');
const input = document.getElementById('todo-input');
const list = document.getElementById('todo-list');

let todos = [];

form.addEventListener('submit', function(e) {
  e.preventDefault();
  const text = input.value.trim();
  if (!text) return; // tránh thêm task rỗng
  todos.push(text);
  input.value = '';
  renderTodos();
});

function renderTodos() {
  list.innerHTML = ''; // xóa sạch trước khi vẽ lại
  todos.forEach((todo, index) => {
    const li = document.createElement('li');
    li.textContent = todo;

    const delBtn = document.createElement('button');
    delBtn.textContent = 'Xóa';
    delBtn.addEventListener('click', () => {
      todos.splice(index, 1);
      renderTodos();
    });

    li.appendChild(delBtn);
    list.appendChild(li);
  });
}

// Lần đầu load page, nếu muốn khởi tạo danh sách trống
renderTodos();

Giải thích sau khối code

  1. todos là mảng lưu text các task.
  2. Mỗi lần submit, ta kiểm tra input không rỗng rồi push() vào mảng.
  3. renderTodos(): reset nội dung <ul> , loop qua todos tạo từng <li> + nút xóa.
  4. Khi nhấn nút Xóa, ta splice() phần tử ra khỏi mảng rồi gọi lại renderTodos() để cập nhật UI.

Bước 5: Chạy thử và nâng cấp

  • Mở index.html bằng Live Server hoặc trình duyệt thường.
  • Thêm task, xóa task, thử bấm Enter.
  • Nếu ổn rồi thì… nghĩ xem lưu lên localStorage, thêm checkbox đánh dấu hoàn thành, filter theo “done”/“pending” cho next version.
